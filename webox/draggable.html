Draggable组件说明

需要加载jquery.ui.core.min.js，这个文件里是四个jquery官网的文件，分别是： jQuery UI 1.8.4； jQuery UI Widget 1.8.4；jQuery UI Mouse 1.8.4；jQuery UI Sortable 1.8.4主要有用到的是jQuery UI Sortable 1.8.4，但是jQuery UI Sortable 1.8.4需要其他三个支持才能使用。
